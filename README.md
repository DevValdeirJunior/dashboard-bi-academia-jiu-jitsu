# dashboard-bi-academia-jiu-jitsu
Projeto acadêmico desenvolvido com foco em Business Intelligence aplicado a uma academia de Jiu-Jitsu real. O projeto envolve coleta de dados em Excel, processo de ETL no Power Query, modelagem, criação de KPIs, medidas DAX e dashboards no Power BI, com publicação via Gateway para apoio à tomada de decisão gerencial e análise de desempenho.
